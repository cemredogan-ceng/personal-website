Clean Code

Bu yazımızda kod yazan, yazılım ile uğraşan herkesin sıklıkla duyduğu “clean code” kavramından bahsedeceğiz. Clean code nedir? Temiz kod yazmak mümkün müdür? Ekip arkadaşlarının sorunsuz bir şekilde anlayabileceği, açık ve anlaşılır kod yazabilmek için neler yapabiliriz? Clean code yazmazsak neler olur, ne gibi sorunlarla uğraşmak zorunda kalır, bir kodu temiz kod haline getirebilmek için neler yapabiliriz gibi sorulara yanıt arayacağız.
